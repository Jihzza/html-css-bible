Document Structure

  • <!Doctype html> - Defines the document type and version of HTML
  • <html> - Root element that groups all the page content
  • <head> - Contains metadata informations, links for the stylesheets, scripts, fonts and other non visible information
  • <body> - Contains all the visual content

Semantic Layout Elements

  • <header> - Builds the Header where it can contain a logo and some navigation elements
  • <nav> - Navigation links within sections
  • <main> - Main content of the documents. (1 per page)
  • <section> - Groups the content per themes. (eg.: aboutMe, Services, ect)
  • <article> - Groups individual content. More used for articles, testmonials, ect...
  • <aside> - Used to create a sidebar or box that will contain content associated with the main content that it's directed to
  • <footer> - Presented on the bottom of the page. Similar function as the <header>, normally contains links to social medias, map of the website, terms and privacy policies

Text Content

  • <h1> - Used for Titles. Hierarchy is used, <h1> is for main titles, <h2> is for section titles, <h3> is for card titles, etc
  • <p> - Used for normal text

Lists

  • <ul> - Unordered list, appears in bullet points
  • <ol> - Ordered list, appears in numbers
  • <li> - List item, used in between the <ul> and <ol>

Links and Media

  • <a> - Link element, can be images, texts, etc
  • <img> - Embeed images

Forms

  • <form> - Used when a form is created
  • <input> - Input box that always, text, checkout, email, ect...
  • <textarea> - Input box that allows multiple lines of text
  • <button> - Button
  • <label> - Used to give the a name or title
  • <select> - Used to create dropdowns

Figures

  • <figure> - Used to have an element together with a caption. More used with images and videos.
  • <figcaption> - Caption for the <figure>

Tables

  • <table> - Creates a table
  • <th> - Used for table headers
  • <tr> - Used for table row
  • <tbody> - Used for the main content of the table
  • <tfoot> - Used for the footer of the table

Other Common Elements

  • <div> - Non semantic generic container
  • <span> - Used inside text elements to give custom styling to a specific text
  • <strong> - Bold Text
  • <em> - Italic Text
  • <br> - Line Break
  • <hr> - Horizontal Line
  • <iframe> - Embeds other HTML pages
  • <video> - Video files
  • <audio> - Audio files
  • <link> - Links to external resources like fonts and stylings
  • <meta> - Metadata present on <head>
  • <script> - JavaScript code or links

HTML Entities

  • & + lt - less than symbol <
  • & + gt - greater than symbol >
  • & + amp - ampersand symbol &
  • & + quot - double quote "
  • & + nbsp - non visible space
